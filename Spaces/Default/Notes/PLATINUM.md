# PLATINUM Vault Documentation

## About PLATINUM

This repository includes a set of preconfigured plugins to enhance your productivity and workflow. You can remove any that you do not need.

**Getting Started:**

- Install the Roboto Condensed font for optimal display
- Configure your Copilot API key to enable AI features
- Review the pre-installed plugins below and customize as needed

---

## Pre-installed Obsidian Plugins

| Plugin             | Description                                                              | Purpose                                                                                                | Link                                                                 |
| ------------------ | ------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------- |
| **Templater**      | Create and use templates                                                 | Essential for creating dynamic templates with custom scripts and variables                             | [Documentation](https://silentvoid13.github.io/Templater/)           |
| **QuickAdd**       | Quickly add new pages or content to your vault                           | Streamlines the process of adding new notes, tasks, and content through customizable macros            | [Documentation](https://quickadd.obsidian.guide/docs/)               |
| **Dataview**       | Complex data views for the data-obsessed                                 | Query and display data from your notes using a powerful query language                                 | [Documentation](https://blacksmithgu.github.io/obsidian-dataview/)   |
| **Copilot**        | Your AI Copilot: Chat with Your Second Brain, Learn Faster, Work Smarter | AI-powered assistant that helps you interact with your vault, generate content, and enhance your notes | [Documentation](https://github.com/logancyang/obsidian-copilot)      |
| **Style Settings** | Offers controls for adjusting theme, plugin, and snippet CSS variables   | Customize the appearance of your vault without editing CSS files directly                              | [Documentation](https://github.com/mgmeyers/obsidian-style-settings) |

**Configuration:** Most plugins have their configuration stored in `.obsidian/plugins/[plugin-name]/data.json`. Access plugin settings through Obsidian's Settings → Community plugins menu.
